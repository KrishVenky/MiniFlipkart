# MF-67: Logging Policy

## Retention
- Security logs: 7 years
- Audit logs: 5 years
- Application logs: 90 days
- Access logs: 1 year

## Redaction
- PII fields: Email, phone, address
- Payment data: Card numbers, CVV
- Authentication: Passwords, tokens

## Access
- Security team: Full access
- Compliance: Read-only
- Developers: Application logs only
- Automated: Monitoring systems
